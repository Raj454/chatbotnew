# Craffteine Admin - Quick Reference Card

## Access
**URL:** `yoursite.com/admin`

---

## Bot Instructions (How the AI Talks)

### Change Greeting
Find `**GREETING:**` → Edit text → Save

### Add New Rule
Find `**RULES:**` → Add `- Your new rule` → Save

### Add Promotion
Add new section:
```
**PROMOTION:**
Mention: Free shipping on orders over $50!
```
(Delete when promotion ends)

---

## Ingredients (What AI Recommends)

### Mark Out of Stock
Ingredients → Find item → Toggle OFF (gray)

### Mark Back in Stock
Ingredients → Find item → Toggle ON (purple)

### Add New Ingredient
Fill form → Select blend → Click Add

---

## Flavors (Stick Pack Only)

### Mark Out of Stock
Flavors → Find item → Toggle OFF

### Add New Flavor
Type name → Click Add

---

## Changes Take Effect
- **Inventory changes:** Instant
- **Bot instructions:** Within 60 seconds
- **Always start NEW chat to test**
